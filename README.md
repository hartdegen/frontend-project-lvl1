Here is a collection of five mini-games:
1. "Is a prime?"
2. "Calculator"
3. "The Greatest Common Divisor"
4. "Arithmetic progression"
5. "Parity Check"

Each of the games gives you a small math quest, but do not run right after the calculator;)
